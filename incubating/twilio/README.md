## Main arguments
- `TWILIO_SID`
- `TWILIO_TOKEN`
- `TWILIO_PHONE_FROM`
- `TWILIO_PHONE_TO`
- `TWILIO_TYPE` - type of message [build, message, hello]

For **message** type you must provide `TWILIO_MESSAGE` argument

## Config for codefresh.yml
```
version: '1.0'
...
steps:
  ...
  TestSMS:
    type: twilio
    arguments:
      TWILIO_SID: ID
      TWILIO_TOKEN: ${{TWILIO_TOKEN}}
      TWILIO_PHONE_FROM: ${{TWILIO_PHONE_FROM}}
      TWILIO_MESSAGE: 'my message'
  ...
...
```
